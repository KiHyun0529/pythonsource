[13. 함수의 매개변수(Parameter)와 인수(Argument)]

매개변수(Parameter) : 함수를 정의할 때 값을 받기 위해 적는 변수

가변 매개변수(Variable-length Parameter) : 인수의 개수가 정해지지 않았을 때 여러 값을 받는 매개변수

인수(Argument) : 함수를 호출할 때 실제로 전달하는 값

def test(a, *args):
    print(a)
    print(args)

test(10, 20, 30, 40)

a : 일반 매개변수

*args : 가변 매개변수

10, 20, 30, 40 : 인수

==============================
시험에 자주 쓰는 함수
==============================

[문자열(str)]

len() : 문자열의 길이(문자 개수)를 반환
예) len("Hello") → 5

upper() : 문자열을 모두 대문자로 변환
예) "hello".upper() → "HELLO"

lower() : 문자열을 모두 소문자로 변환
예) "HELLO".lower() → "hello"

replace() : 문자열의 특정 내용을 다른 내용으로 변경
예) "apple".replace("a", "A") → "Apple"

find() : 문자열에서 특정 문자의 위치(인덱스)를 반환
(없으면 -1 반환)
예) "apple".find("p") → 1

count() : 특정 문자가 몇 개 있는지 반환
예) "apple".count("p") → 2

split() : 문자열을 나누어 리스트로 반환
예) "a b c".split() → ['a', 'b', 'c']

strip() : 문자열의 앞뒤 공백을 제거
예) " hello ".strip() → "hello"


==============================

[리스트(list)]

append() : 리스트의 맨 뒤에 요소 추가
예) list.append(10)

extend() : 여러 요소를 한 번에 추가
예) list.extend([4,5])

insert() : 원하는 위치에 요소 삽입
예) list.insert(1, 100)

remove() : 지정한 값을 삭제
예) list.remove(3)

pop() : 마지막 요소를 꺼내면서 삭제
(인덱스를 지정하면 해당 위치 삭제)
예) list.pop()

sort() : 오름차순 정렬
예) list.sort()

reverse() : 리스트의 순서를 뒤집기
예) list.reverse()

clear() : 리스트의 모든 요소 삭제
예) list.clear()


==============================

[딕셔너리(dict)]

keys() : 모든 키를 반환
예) dic.keys()

values() : 모든 값을 반환
예) dic.values()

items() : (키, 값) 형태로 반환
예) dic.items()

get() : 키에 해당하는 값을 반환
(키가 없으면 None 반환)
예) dic.get("name")

update() : 딕셔너리에 값을 추가하거나 수정
예) dic.update({"age":20})

pop() : 지정한 키와 값을 삭제하고 값을 반환
예) dic.pop("name")


==============================

[집합(set)]

add() : 요소 하나 추가
예) s.add(10)

remove() : 요소 삭제
(없으면 오류 발생)
예) s.remove(10)

discard() : 요소 삭제
(없어도 오류 없음)
예) s.discard(10)

union() : 두 집합을 합침(합집합)
예) a.union(b)

intersection() : 두 집합의 공통 요소 반환(교집합)
예) a.intersection(b)

difference() : 다른 집합에 없는 요소 반환(차집합)
예) a.difference(b)


========================================

[시험 단골 문법]

if : 조건이 참(True)일 때 실행

elif : 앞의 조건이 거짓일 때 다른 조건 검사

else : 모든 조건이 거짓일 때 실행

while : 조건이 참인 동안 반복

for : 반복 가능한 객체를 순서대로 반복

range() : 일정 범위의 숫자를 생성

enumerate() : 인덱스와 값을 함께 반환

break : 반복문을 즉시 종료

continue : 현재 반복을 건너뛰고 다음 반복 실행

def : 함수를 정의

return : 함수의 결과를 반환하고 종료

input() : 사용자에게 입력을 받음

print() : 화면에 출력

*args : 여러 개의 위치 인수를 tuple로 받음

**kwargs : 여러 개의 키워드 인수를 dict로 받음

len() : 길이(개수)를 반환

type() : 자료형을 확인

ord() : 문자를 아스키(유니코드) 코드 값으로 변환

chr() : 아스키(유니코드) 코드 값을 문자로 변환

list() : 리스트를 생성하거나 리스트로 변환

tuple() : 튜플을 생성하거나 튜플로 변환

dict() : 딕셔너리를 생성하거나 딕셔너리로 변환

set() : 집합을 생성하거나 집합으로 변환

int() : 정수형으로 변환

float() : 실수형으로 변환

str() : 문자열로 변환

bool() : True 또는 False로 변환

sum() : 반복 가능한 객체의 모든 요소를 더한 값을 반환

lambda : 이름 없이 한 줄로 만드는 익명 함수

f-string : 문자열 안에 변수나 계산식을 쉽게 출력하는 방법
